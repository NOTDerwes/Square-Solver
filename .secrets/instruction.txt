Чтобы решить квадратное уравнение, приведите исходное уравнение к виду:
    ax^2 + bx + c = 0, где a, b, c - какие-то числа (если в приведенном виде степень многочлена больше 2, обратитесь к интернету).

Если у Вас получилось линейное уравнение (коэффициент a = 0), решите его самостоятельно, этому учат в 3м классе

Посчитайте дискриминант (D) уравнения по формуле D = b^2 - 4ac.
    Если дискриминант меньше 0, то уравнение имеет 0 корней (на самом деле нет, но об этом вам расскажут как-нибудь потом).

    Если дискриминант равен 0, уравнение имеет ровно 1 корень, который вычисляется по формуле x = -b / (2a).

    Если дискриминант больше 0, уравнение имеет 2 корня.
        Меньший корень вычисляется по формуле x_м = (-b - sqrt(D)) / (2a).
        Больший корень вычисляется по формуле x_б = (-b + sqrt(D)) / (2a).
